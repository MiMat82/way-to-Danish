
System forudsætninger:
--------------------------------------------------------------------------------------
For at bruge denne cd skal du have en PC med:
- Cd-rom drev, lydkort med højttalere/headset
- Microsoft Internet Explorer 4.x eller nyere
- Microsoft Windows MediaPlayer 5.x eller nyere


Start cd'en:
--------------------------------------------------------------------------------------
- Indsæt cd'en i cd-drevet
- Dialogboksen: 'Automatisk afspilning' kommer nu frem på skærmen
- Vælg: 'Åbn mappe for at vise filer'
- Marker filen: 'index/index.htm' og tryk 'Return' eller dobbeltklik på filen.

Hvis dialogboksen 'Automatisk afspilning' ikke kommer frem på skærmen:
- Åbn: 'Denne computer'/'Computer'
- Åbn: cd-drevet/dvd-drevet (drev D: eller E: eller F:) 
- Marker filen: 'index/index.htm' og tryk: 'Return' eller dobbeltklik på filen.


Tekniske informationer:
-----------------------
Denne CD indeholder html- og Windows Media filer.

Audioformat: 
- Window Media ASF audio
- File extension:  *.asf

Anbefalet min.:
200Mhz Processor, 32MB RAM, 4MB grafik kort.

CD-ROM format:
ISO 9660, læsbar på alle systemer.
